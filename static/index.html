Joining a Game

  • This is a multiplayer game, you must have at least one other person to play with (up to a maximum of 6).
  • If you are the first player, click the Create Game button to generate a lobby ID.
  • Communicate this ID to the other people you would like to play with so they can join.
  • TIP: tap the button again to copy the code to your clipboard.
  • Once you have a lobby ID, input it into the Lobby ID field, make a name and click Join Game.

How to Play

  • Pick a 5 letter word, the other players will try to guess this word.
  • Once you have picked your word, click Set Word (or enter on your keyboard) to ready up.
  • When everyone is ready, click Start Game to start.
  • Once in game, tap on a player's name to start guessing their word.
  • After a guess letters that return green are the correct letter in the correct spot, letters that return yellow are the correct words in an incorrect spot.
  • The game ends when one player has correctly guessed everyone else's words.

Scoring

  • Each player's word can give a max score of 1000.
  • Each grey letter guessed subtracts 50 from the score, and each yellow subtracts 25. Green letters do not subtract from the score.
  • Each correctly guessed word adds to your total score.
  • TIP: The game ends as soon as one player finishes all words. In small games such as those with only two players, its important to guess quickly to be the first to finish as the second player will not receive any score.
